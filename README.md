# Number Guessing Game – Java

## Overview
This is a simple console-based Number Guessing Game written in Java.  
The program randomly selects a number within a predefined range, and the player tries to guess it. After each guess, the game gives feedback indicating whether the guess is too high, too low, or correct.

## Features
- Random number generation  
- Console-based user input  
- Real-time feedback for each guess  
- Tracks the total number of attempts  
- Option to play multiple rounds  

## Technologies Used
- Java (JDK 8 or higher)  
- Scanner class for reading user input  
- Random class for number generation  

## How to Run
1. Open the project folder in any Java IDE (such as IntelliJ, Eclipse, or VS Code).  
2. Open the file `NumberGuessingGame.java`.  
3. Run the program using the IDE’s built-in run option.
